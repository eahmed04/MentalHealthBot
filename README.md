
# Mental Health Counseling Assistant

A personal AI project focused on building a supportive, empathetic conversational assistant for mental health–related discussions.

## Overview
This project explores how structured prompts and role-based message design can guide large language models to produce emotionally aware, safety-conscious responses.

## Key Features
- Role-based conversation design (System, User, Assistant)
- Mental health–specific prompt engineering
- JSONL data preparation for GPT-based models
- Safety-aware response guidelines

## Tech Stack
- Python
- OpenAI API
- Jupyter Notebook
